- [x] Flutter CLI
    - flutter doctor -> Verificar se o ambiente está pronto para o desenvolvimento
    - flutter create {project-name}-> Cria um projeto padrão
    - flutter run -> Compila e executa o ambiente de desenvolvimento 
    - flutter build apk / flutter build appbundle -> Compila para a versão de produção
    - flutter analyze -> Analisa o código em busca de erros
    - flutter pub get -> Atualiza todas as dependências listadas em [pubspec.yaml]
- [x] FVM (opcional)
    - fvm list - Lista todas as versões disponíveis
    - fvm install 3.44.6 -> Instalar a versão específica
    - fvm use {version} -> Fixa a a versão para o projeto
    - fvm flutter create . --platforms android --org {dominio} --project-name {name} 
        -> Cria o projeto flutter para android e configura o domínio
